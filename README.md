# Customer Shopping Behavior Analysis

## 📌 Project Overview
An end-to-end data analysis project using **Python** for cleaning, **SQL** for deep-dive analysis, and **Power BI** for visualization.

## 🛠️ Tools Used
* **Python (Pandas):** Data cleaning and transformation.
* **SQL (SQLite):** Complex querying and aggregation.
* **Power BI:** Interactive dashboarding.
* **Gamma:** Project presentation.

## 📂 Folder Structure
* `scripts/`: Python ETL scripts.
* `sql/`: SQL analysis files.
* `dashboard/`: Power BI .pbix file.
* `presentation/`: PDF version of the Gamma PPT.

## 🚀 How to Run
1. Run `scripts/load_to_sqlite.py` to process the data.
2. Open `sql/analysis_queries.sql` to view the analysis.
3. Open `dashboard/shopping_report.pbix` in Power BI.
